DATOS DE UTILIDAD

Todos los valores están expresados en días hábiles.

S: Semana

M: Mes

A: Año

Para facilitar la visualización de los datos, se grafica una muestra de los datos totales. Cada gráfico mostrará aproximadamente 90 datos correspondientes a la variable en el plazo de tiempo seleccionado siguiendo la siguiente regla:

Para variables con muestras diarias:

  • 1S: se toman todos los datos de la semana.
  • 1M: se toman todos los datos del mes
  • 3M: se toman todos los datos de los últimos tres meses.
  • 6M: se toma una muestra de 90 datos (aprox.) de los últimos seis meses, con una diferencia de dos días entre sí.
  • 1A: se toma una muestra de 90 datos (aprox.) del último año, con una diferencia de cuatro días entre sí.
  • 2A: se toma una muestra de 30 datos (aprox.) de los últimos dos años, con una diferencia de ocho días entre sí.
  • 5A: se toma una muestra de 90 datos (aprox.) de los últimos cinco años, con una diferencia de veinte días entre sí.
  • 10A: se toma una muestra de 90 datos (aprox.) de los últimos diez años, con una diferencia de cuarenta días entre sí.

Para variables con muestras mensuales:

  • En los intervalos 1S, 1M, 3M, 6M, 1A y 2A se toman todos los datos disponibles.
  • 5A: se toma una muestra de 90 datos (aprox.) de los ultimos cinco años, con una diferencia de cuatro meses entre sí.
  • 10A: se toma una muestra de 90 datos (aprox.) de los últimos diez años, con una diferencia de cuatro meses entre sí.
